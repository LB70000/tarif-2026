# Tarif 2026 — Le Besson

Configurateur interactif de tarif plaquettes d'identification forestière.

## Contenu

Outil web autonome (fichier HTML unique) permettant à un utilisateur de naviguer
dans le tarif 2026 via un formulaire guidé en 4 étapes :

1. Univers de traçabilité (forestier, papetier, bois scié)
2. Caractéristique de marquage (alphanumérique, code-barres, logo, QR code)
3. Format de plaquette
4. Sélection de la ou des références

## Utilisation

Ouvrir `index.html` dans un navigateur. Aucune dépendance externe, aucun serveur requis.

## Gammes couvertes

- Plaquettes standard — UNIVERS (jet d'encre) & HORIZON (laser)
- Plaquettes qualité papetière — BIOTAG (laser)
- Plots pour bois scié (laser)
